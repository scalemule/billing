# @scalemule/billing

React billing components for ScaleMule applications, built on Stripe Elements.

## Install

```bash
npm install @scalemule/billing @stripe/stripe-js @stripe/react-stripe-js
```

## Peer dependencies

- `react >= 18`
- `react-dom >= 18`
- `@stripe/stripe-js ^5`
- `@stripe/react-stripe-js ^3`

## Exports

- `BillingProvider`
- `PaymentElement`
- `SetupElement`
- related TypeScript types
